# Free Invoice Generator

**Live:** https://free-invoice-generator.onrender.com/

An invoice generator that is free and gives you a downloadable PDF. No hassles, no frills.

Inspired by the millions of "Free Invoice Generators" that have usage limits, make you make an account, or are deceptively trying to funnel you to their ERP software.

## Running locally

```bash
npm install
npm start
```

Then open http://localhost:3000 in your browser.

For development with auto-reload:

```bash
npm run dev
```
